What is cryptography about ?

Cryptography

Generally, cryptography is about constructing and analyzing protocols that prevent third parties or the public from reading private messages or data.
Modern cryptography is devided into 2 parts : Symmetric and Asymmetric cryptography.

Symmetric cryptography

Symmetric cryptography system is a system which uses the same cryptographic key for both encryption of plaintext and decryption of ciphertext.

Asymmetric cryptography

Asymmetric cryptography system is a system which uses pairs of keys: public keys (which may be known to others) and private keys (which may never be known by anyone except the owner).

Algorithms

Camellia

- created in 2000 by colaboration of MEC a NTT
- uses symmetric key
- uses fixed block size of 128 bits
- key size of 128, 192 and 256 bits

Kuznyechik

- created in 2015 as a part of GOST standard in addition to Magma cipher
- uses symmetric key
- uses fixed block size of 128 bits
- key size of 256 bits

ChaCha

- created in 2008 by D. J. Bernstein as improved Salsa cipher
- stream cipher
- works on 512 bit long inputs
- uses 256 bit key
