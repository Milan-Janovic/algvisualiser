Encryption process consists of 20 rounds alternating between column and diagonal rounds.
In column round function we modify columns of matrix separatively and in diagonal round we modify diagonals of matrix separatively both using so called quater round function, which takes four 32-bit long inputs (a,b,c,d) from matrix and returns four modified 32-bit outputs as defined :
a = (a + b) % 32
d = d XOR a
d = d << 16
c = (c + d) % 32
b = b XOR c
b = b << 12
a = (a + b) % 32
d = d XOR a
d = d << 8
c = (c + d) % 32
b = b XOR c
b = b << 7
Return a, b, c, d
As previously mentioned encryption process consists of 20 rounds alternating between column and diagonal rounds using quater round four times in each round. So after 10 column rounds and 10 diagonal rounds we get final matrix (FM). To obtain CT we XOR corresponding parts of FM and PT matrix (PTM), which was created by splitting PT into 32-bit long parts.
